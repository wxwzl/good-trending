# 爬虫商品抓取数据逻辑：

读取apps\crawler\src\category\categoryKeys.json 文件中的亚马逊类目数组数据，到内存中
以该数组项作为关键词到google上搜索：

## 搜索规则：

（google上的日期2026-03-08 相当于2026-03-08 00：00：00）

- reddit平台：`site:reddit.com 类目名称 after:2026-03-08 before:2026-3-9`
- x 平台：`site:x.com 类目名称 after:2026-03-01 before:2026-3-2`

## 搜索今天一天的数据，每各2小时爬取一次。

### 规则如下：

- reddit平台：`site:reddit.com 类目名称 after:[今天日期（格式：2026-03-09）]`

第一步：获取帖子的总数有多少，，更新 类目热度表，今天的搜索结果总数。
第二步：根据google搜索到的结果，挨个爬取结果连接里的内容，最多只找出前10个亚马逊上的商品链接，只能最多挨个爬取30个搜素结果的内容，爬取的商品存在内存中或者写入临时的json文件。
第三步，根据第二步爬取到的商品，以该商品在亚马逊上的名称为关键词进行google搜索reddit站点，查看搜索结果的总条数,总条数数据所在的dom结构如下：

```
<div id="result-stats">About 19,000 results<nobr> (0.70s)&nbsp;</nobr></div>

```

## 搜索昨天一天的数据：每天凌晨执行。

### 规则如下：

- reddit平台：`site:reddit.com 类目名称 after:[昨天日期（格式：2026-03-09）] before:[今天日期（格式：2026-03-09）]`

### 搜索步骤

第一步：获取帖子的总数有多少。（将该数据插入到类目热度表中，插入数据：搜索的类目名称，贴子总数，数据抓取的日期）
第二步：根据google搜索到的结果，挨个爬取结果连接里的内容，找出前10个亚马逊上的商品链接，只能挨个爬取30个搜素结果的内容，
这一步把爬取到的商品信息(见下方的商品表)插入到商品表中（如果存在就跳过），并更新商品趋势因素统计表中 `近7天出现的次数`,` 近15天出现的次数`,`近30天出现的次数`,`近60天出现的次数`的值。并保存刚才搜索到的商品的ID到内存中。
第三步，把刚才搜到的商品以商品名称为关键词进行google搜索reddit站点，按本周，本月 近15天，近30天，近60 天 分别查看搜索结果的总条数，总条数数据所在的dom结构如下：

```
<div id="result-stats">About 19,000 results<nobr> (0.70s)&nbsp;</nobr></div>

```
