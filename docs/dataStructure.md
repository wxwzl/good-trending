# 爬虫商品抓取数据逻辑：

按亚马逊上的类目的名称作为关键词到google上搜索：

## 搜索规则：

（google上的日期2026-03-08 相当于2026-03-08 00：00：00）

- reddit平台：`site:reddit.com 类目名称 after:2026-03-08 before:2026-3-9`
- x 平台：`site:x.com 类目名称 after:2026-03-01 before:2026-3-2`

## 搜索今天一天的数据

搜索昨天一天的数据：

- reddit平台：`site:reddit.com 类目名称 after:2026-03-01 before:2026-3-2`
- x 平台：`site:x.com 类目名称 after:2026-03-01 before:2026-3-2`
  第一步：获取帖子的总数有多少。（将该数据插入到类目热度表中，插入数据：搜索的类目名称，贴子总数，数据抓取的日期）
  第二步：根据google搜索到的结果，挨个爬取结果连接里的内容，找出前10个亚马逊上的商品链接，只能挨个爬取30个搜素结果的内容，
  这一步把爬取到的商品信息(见下方的商品表)插入到商品表中（如果存在就跳过），并更新商品出现次数统计表。

- 类目热度表

- 商品表：
- - 商品名称：商品售卖平台标注的商品名称
- - 商品描述：商品售卖平台标注的商品描述
- - 商品价格：商品售卖平台标注的价格
- - 商品连接：商品售卖平台的连接
- - 商品ID:商品售卖平台的ID，唯一
- - slug: 根据商品名称生成唯一，重复时+[-数字]，尝试10次，超过10次，采用添加商品ID后缀[-商品ID]

- 商品出现次数统计表
- - 商品ID
- - 上一次出现的时间
- - 更新时间
- - 创建时间
- - 近7天出现的次数，采用滑动窗口存储，是一个字符串："0010101",最多7位数字，每次更新都要取原来的字符串，根据前一天日期-上一次出现的时间-1，按日期相减，大于0就往字符后面加几个0，然后取后7位数字更新该值，如

````
前一天日期：2026-3-8，上一次出现的时间：2026-3-5，当前值："0001101"，那么前一天日期-上一次出现的时间-1=2，字符串先变成
"0001101"+"001"="0001101001",最后的更新值"0001101001".substr(str.length - 7);，然后然后将前一天日期填入上一次出现的时间的列中。

```。
````
